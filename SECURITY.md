# Security Policy

Builder OS controls local model processes and coding-agent sessions. Treat
changes to command execution, permissions, project paths, and configuration as
security-sensitive.

## Reporting A Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories:

<https://github.com/IsmailHachimi/builder-os/security/advisories/new>

Do not include secrets, private source code, or sensitive local logs in public
issues.

## Scope

The current supported environment is Apple Silicon macOS using the latest
released Builder OS version. Third-party vulnerabilities in OpenCode, MLX-LM,
models, Homebrew, or generated project dependencies should also be reported to
their respective maintainers.
