# Strict Local Workstation Rules

- All inference must use the configured MLX endpoint on `127.0.0.1`.
- Do not use web search, web fetch, remote MCP servers, or cloud providers.
- Do not expose project contents, prompts, diffs, or session data externally.
- Treat dependency installation and git publishing as approval-gated actions.
- Work autonomously inside the current project after presenting a plan, except
  when the plan identifies broad scope, ambiguity, or a gated action.
