# Third-Party Software

Builder OS is an independent MIT-licensed project. It installs, invokes, or
integrates with external software, but does not bundle that software's source
code in this repository.

## Core Integrations

- [OpenCode](https://github.com/anomalyco/opencode) is used as the coding-agent
  interface and execution engine. OpenCode is distributed under the MIT
  License. Builder OS is not affiliated with or endorsed by the OpenCode team.
- [MLX-LM](https://github.com/ml-explore/mlx-lm) provides local model inference.
  It remains subject to its own license and notices.
- [vLLM](https://github.com/vllm-project/vllm) provides local model inference
  on supported Linux systems and retains its own license and notices.
- Homebrew, Python, Zsh, `jq`, launchd, systemd, NVIDIA software, and other
  system tools remain subject to their respective licenses.

## Models And Generated Projects

Model weights are downloaded separately and remain subject to each model
publisher's license. Review a model's license before downloading or
redistributing it.

Projects generated by `bos init` install third-party package dependencies.
Those packages retain their own licenses. Builder OS's MIT License applies only
to Builder OS source code and does not replace third-party terms.
